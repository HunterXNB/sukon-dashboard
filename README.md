# Sukon Admin Dashboard

## Overview

Sukon Admin Dashboard is a web-based admin panel designed to manage the Sukon platform, which connects doctors with patients. The dashboard provides administrators with essential tools to manage users, roles, and app settings efficiently.

## Features

### ✅ Authentication

- Secure login and logout functionality
- Forgot password and reset password logic
- Change password feature for enhanced security

### 📊 Dashboard Home

- Displays key platform statistics for better insights

### 🛠️ Management Tables

- **Roles Management** – Manage different user roles and permissions
- **Admins Management** – Handle admin users with appropriate access control
- **Tiers Management** – Manage subscription or service tiers
- **Users Management** – View and manage platform users

### ⚙️ App Settings

- A dedicated form to modify platform settings

### 🌍 Multi-Language Support

- Supports **English** and **Arabic** for a better user experience

### 🌗 Theme Support

- **Dark Mode** and **Light Mode** toggle for accessibility and user preference
